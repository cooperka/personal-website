# cooperka.com

My [personal website](https://cooperka.com).

## Developing

1. `yarn install`
2. `yarn start`
   - It will automatically open <http://localhost:3000/> in your browser
3. `yarn test`

## Publishing

Hosted on [Netlify](https://app.netlify.com/projects/cooperka/overview).

1. `yarn run deploy`
